# SocietalProblems

Research covering LLMs ability to help address societal problems. 


Initial prompt to generate lists of societal problems that AI may be able to address.

Can you analyze the top 100 papers on arxiv and other popular white paper websites that propose a conjecture or an unsolved problem and rank them in terms of benefit to society if proved or solved. The ranking should include how likely an LLM would be to solve them. So probably a score for impact to humanity, how much benefit to society the problem fixes. and a ratio of something like benefit to society over how likely an LLM such as yourself would be to solve them or make significant progress.  There should also be a link to the paper, and a short paragraph detailing the problem and how society could benefit. The format should be in csv so please avoid commas in the paragraph.





After research was concluded, a the LLM most involved in solving the solution was asked to create a methodology paper using the prompt below.

Are you able to create a new tex file that covers the methodology we used to get to this step. Include the important papers and discoveries that shifted the thought process. Please include a section on how humans can improve their own research practices based on your own methodology vs standard AI research. You can paste it a text field like you've done with the previous tex files. The target audience are AI enthusiasts and the scientific community especially in AI safety.
